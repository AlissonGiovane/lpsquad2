PORTFÓLIO

JOGO DO PONG

O primeiro projeto que fizemos, individualmente, foi recriar o famoso jogo “Pong” de duas maneiras diferentes. Primeiramente, para programar esse jogo clássico ultilizamos a plataforma “Scratch”, em que o principal objetivo era entender a lógica de programação. Logo após, recriamos o jogo novamente ultilzando a linguagem JavaScript. Aprendemos a ultilizar variáveis e funções. Criamos o jogo completo, ou seja, com animações, controles, sons e placar.

Feito com: Lógica de programação JavaScript

JOGO DO FREEWAY

Após o pong, criamos do zero o famoso jogo "Freeway'', com animações, pontuações e sons. Ao desenvolver o código desse jogo aprendemos como usar listas (arrays) e como evitar código duplicado com laços de repetição.

Feito com: Lógica de programação JavaScript

PROJETO EM JAVASCRIPT

A fim de aprender mais sobre JS, fizemos uma página em que é possível captar interações do usuário, validar formulários e calcular o IMC. Ao desenvolver esse programa, juntamente com os instrutores da plataforma “Alura”, conhecemos algumas das chamadas boas práticas de organização de código

Feito com: Lógica de programação JavaScript

BARBEARIA ALURA

A fim de conhecer melhor sobre HTML e CSS, desenvolvemos uma página para a fictícia “Barbearia Alura”. Nesse projeto, aprendemos a definir estilos para elementos usando CSS, inline e block, lide com bordas e pseudo-classes no CSS. Além disso, conhecemos o reset.css e a hierarquia no CSS.

Feito com: HTML e CSS
